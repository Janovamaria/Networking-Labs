# TCP/IP Model

## Objective

To understand the TCP/IP (Transmission Control Protocol/Internet Protocol) Model and how it enables communication over computer networks.

## TCP/IP Layers

### Application Layer

* Provides network services to applications.
* Protocols: HTTP, HTTPS, FTP, SMTP, DNS, SSH

### Transport Layer

* Provides end-to-end communication.
* Protocols:

  * TCP (Reliable communication)
  * UDP (Fast communication)

### Internet Layer

* Responsible for logical addressing and routing.
* Protocols:

  * IP
  * ICMP
  * ARP

### Network Access Layer

* Handles communication between devices on the same network.
* Technologies:

  * Ethernet
  * Wi-Fi

## Difference Between TCP and UDP

| TCP                                | UDP                              |
| ---------------------------------- | -------------------------------- |
| Connection-oriented                | Connectionless                   |
| Reliable                           | Faster                           |
| Error checking                     | Minimal error checking           |
| Used for Web, Email, File Transfer | Used for Streaming, VoIP, Gaming |

## Why the TCP/IP Model is Important

* Forms the foundation of the Internet.
* Enables communication between different networks.
* Helps troubleshoot networking issues.
* Used in almost all modern computer networks.
