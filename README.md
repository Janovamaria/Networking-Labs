# Networking-Labs

## About

This repository contains my networking practice, notes, and hands-on lab documentation. It covers networking fundamentals, protocols, troubleshooting, and practical configurations learned through academic coursework and self-learning.

## Topics Covered

* OSI Model
* TCP/IP Model
* IPv4 Addressing
* Subnetting
* DNS
* DHCP
* Routing & Switching
* VLAN Fundamentals
* NAT
* VPN
* Network Troubleshooting
* ACL Basics
* Network Commands (ping, tracert, ipconfig, netstat, nslookup)

## Goal

To document my networking knowledge and practical learning while preparing for roles such as Network Support Engineer, NOC Engineer, System Administrator, and IT Support Engineer.
