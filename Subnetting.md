# Subnetting

## Objective

To understand subnetting, why it is used, and how it helps improve network efficiency and management.

## What is Subnetting?

Subnetting is the process of dividing a large network into smaller, manageable subnetworks (subnets). It helps improve network performance, security, and efficient utilization of IP addresses.

## Advantages of Subnetting

* Reduces network congestion
* Improves network performance
* Enhances security
* Efficient use of IP addresses
* Simplifies network management

## Key Terms

### Network Address

Identifies the network to which a device belongs.

### Host Address

Identifies a specific device within a network.

### Broadcast Address

Used to send data to all devices within a subnet.

### Subnet Mask

Separates the network portion from the host portion of an IP address.

Example:

* IP Address: 192.168.1.10
* Subnet Mask: 255.255.255.0

## Why Subnetting is Important

* Organizes large networks into smaller segments.
* Improves network efficiency.
* Reduces unnecessary traffic.
* Makes troubleshooting easier.
* Widely used in enterprise networks.
